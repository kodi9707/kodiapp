Welcome to My Short Project

Instructions

  • Create a basic web application using Python and Django that includes:
    • A homepage displaying a welcome message.
    • At least one additional page with some functionality (e.g., a form submission page).
    • Set up a Git repository to store your web application code.
    • Create a Dockerfile to containerize your web application.
  • Configure Jenkins to automate the build and deployment process using a Jenkins pipeline.
    • Your Jenkins pipeline should include stages for:
      • Checking out the code from the Git repository.
      • Building the Docker image.
      • Running tests (optional but recommended).
      • Pushing the Docker image to a Docker registry.
      • Deploying the application on Tomcat running in Docker containers.
  • Document the steps required to set up and run the project.
